# Face-Tracking
. Python 3.8.5
. pip 20.2.3
. opencv 4.4.0.44
. pyserial 3.4
. numpy 1.19.1

## Install requirement file with command
* pip install -r requirements.txt

## control pan tilt servo arduino via serial comunnication by face tracking opencv
* Material Required:
  - Arduino Uno Board
  - Pan Tilt Servo & 2 x Servo SG90
  - Jumper Wires
  - USB Cable
